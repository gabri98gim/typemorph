# TypeMorph | Variable Font Lab 🧪

An interactive explorer for **Variable Fonts** built with pure JavaScript and a premium aesthetic. Experiment with over 35+ Google Variable Fonts or upload your own files to discover the power of fluid typography.

[Español 👇](#es-typemorph--laboratorio-de-fuentes-variables)

---

## ✨ Features
- **35+ Built-in Fonts**: A curated catalog of the best variable fonts from Google Fonts.
- **Custom Upload**: Support for `.otf`, `.ttf`, `.woff`, and `.woff2` files with automatic axis detection (fvar table parsing).
- **Dynamic Controls**: Real-time sliders for axes like Weight, Width, Slant, Optical Size, and more.
- **Live Preview System**: Font size control and smooth transitions between typefaces.
- **Visual Dropdown**: Preview each font in the catalog with its actual design before selecting.
- **Advanced Export**: One-click CSS generation with `font-variation-settings`.
- **URL Sharing**: Your current configuration is automatically synced to the URL. Copy and share!
- **Power User Shortcuts**: `Ctrl + Shift + C` to copy CSS instantly.
- **Modern UI**: Full Dark/Light mode support with a focus on minimalism and micro-interactions.
- **No Dependencies**: Built using zero external libraries (Pure Vanilla JS + Tailwind CSS).

## 🛠️ Tech Stack
- **HTML5 & Semantic SEO**
- **Tailwind CSS** (Styling & Dark Mode)
- **Vanilla JavaScript** (Logic, UI, and Font Parsing)
- **FontFace API** (On-the-fly font registration)
- **Binary Font Parsing**: Manual parsing of the `fvar` and `name` tables from font binaries.

## 🚀 Getting Started
1. Clone the repo.
2. Run `npm install`.
3. Start the dev server with `npm run dev`.
4. Open the app and start morphing!

# ES | TypeMorph | Laboratorio de Fuentes Variables 🧪

Un explorador interactivo para **Fuentes Variables** construido con JavaScript puro y una estética premium. Experimenta con más de 35 fuentes variables de Google o sube tus propios archivos para descubrir el poder de la tipografía fluida.

## ✨ Características
- **Más de 35 Fuentes Integradas**: Un catálogo seleccionado de las mejores fuentes variables de Google Fonts.
- **Subida Personalizada**: Soporte para archivos `.otf`, `.ttf`, `.woff` y `.woff2` con detección automática de ejes (parseo de tabla fvar).
- **Controles Dinámicos**: Sliders en tiempo real para ejes como Weight (Peso), Width (Ancho), Slant (Inclinación), Optical Size (Tamaño Óptico) y más.
- **Sistema de Previsualización**: Control de tamaño de fuente y transiciones suaves al cambiar de tipografía.
- **Dropdown Visual**: Previsualiza cómo se ve cada fuente del catálogo antes de seleccionarla.
- **Exportación Avanzada**: Generación de CSS con un solo clic incluyendo `font-variation-settings`.
- **Compartir por URL**: Tu configuración se sincroniza automáticamente en la URL. ¡Copia y comparte!
- **Atajos de Teclado**: `Ctrl + Shift + C` para copiar el CSS al instante.
- **Interfaz Moderna**: Soporte completo de modo Oscuro/Claro centrado en el minimalismo.
- **Sin Dependencias**: Construido sin librerías externas (JavaScript Puro + Tailwind CSS).

## 🛠️ Tecnologías
- **HTML5 & SEO Semántico**
- **Tailwind CSS** (Estilos y Modo Oscuro)
- **Vanilla JavaScript** (Lógica, UI y Parseo de Fuentes)
- **FontFace API** (Registro de fuentes al vuelo)
- **Parseo Binario**: Procesamiento manual de las tablas `fvar` y `name` de los binarios de las fuentes.

## 🚀 Comenzar
1. Clona el repositorio.
2. Ejecuta `npm install`.
3. Inicia el servidor de desarrollo con `npm run dev`.
4. ¡Abre la app y empieza a experimentar!
